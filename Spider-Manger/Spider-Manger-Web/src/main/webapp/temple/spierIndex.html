爬虫电影项目简介

基于Httpclicent Jsoup Jquery Bootstrap
Spring SpringMVC Mybatis Echart 的简单系统
根据需求爬取网络数据，然后进行数据管理
可实现电影信息的模糊查询,删除以及修改
利用echart实现电影分析的数据可视化

爬虫电影项目

项目分为三个模块

第一个模块是爬虫模块负责抓取数据
利用httpclicent对指定链接进行下载
利用jsoup强大的解析功能(css选择器和正则表达式)对数据进行提取

第二个模块是后台模块负责处理请求和数据
使用c3p0管理数据库连接池,并且使用spring对c3p0进行配置
采用MVC模式创建模型和视图和控制器(也就是DAO和Service以及Control)对数据进行分层处理

第三个模块是前台模块负责数据呈现和人机交互
使用boostrap进行布局和构建页面,其中使用CSS自定义一部分样式满足需求
使用Jquery进行事件互动,使得页面具有交互性质
使用ajax进行数据的异步获取,使得降低服务器负担,提供更好的体验
